# My dissertation code
My dissertation aims to: evaluate the stability of current xai models

The code is linked along with the chosen public dataset: CICIDS2017, in the folder called "data"

The first code block creates a concatenated csv file of all the data, the second code blocks created a cleaned and processed csv of all the data

A number of libraries are needed to run this project and include:
- pandas
- numpy
- matplotlib
- scikit-learn
- xgboost
- shap
- lime
- scipy
- jupyter
- notebook

These can be downloaded using the following pip command: pip install pandas numpy matplotlib scikit-learn xgboost shap lime scipy jupyter notebook
